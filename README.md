# RamaTech Solution Website

Modern, single-page DevOps agency landing page built with Next.js, React, Tailwind CSS, and Framer Motion.

## Run Locally

1. Install Node.js LTS (v20+ recommended): [https://nodejs.org](https://nodejs.org)
2. Open terminal in this folder:
   - `cd "c:\Users\ASHISH DWIVEDI\Documents\Cursor_projects\ramtech-website"`
3. Install dependencies:
   - `npm install`
4. Start dev server:
   - `npm run dev`
5. Open:
   - [http://localhost:3000](http://localhost:3000)

## Build for Production

- `npm run build`
- `npm run start`
